JourneyMAPS Documentation
=========================

JourneyMAPS (Journey - A Multistep Automated Parameter Search Package) is a Python package designed for automated parameter optimization and simulation workflows.

The package provides:

* **Journey Package**: Core functionality for automated parameter search and optimization
* **Paths Package**: Implementations of specific paths (e.g., Tidy3D)
